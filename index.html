Core Capabilities

把黑盒通信拆成清晰的工程链路

项目聚焦协议学习与教学演示，覆盖发现、登录、保活、消息、缩略图重组与图像修正等关键路径。

01

在线宣告

周期性发送 OONC、CANC、ANNO，让学生端在同一广播域内发现模拟教师机。

02

登录握手

处理 LOGI 并回送 MESS、LPNT、DMOC，复现教师端建立连接的关键时序。

03

黑屏锁屏

MESS type=0x20 + COMD case 6 双路径实现黑屏安静、键鼠锁定与自动解锁，完整复现教师端控制流程。

04

缩略图重组

接收 LANT/TNAL 分片，拼接 JPEG，并修复 bottom-up DIB 与色度反相问题。

05

信息建档

MESS 0x100000 信息请求，解析计算机名、用户、MAC、OS、CPU、内存及进程/窗口列表。

06

聊天消息

通过 MESS 协议向指定学生发送 UTF-16LE 聊天消息，学生端弹窗显示。

07

远程开关机

COMD 应用命令 cmdId 0x14/0x13 调用学生端 Shutdown.exe，支持强制立即与倒计时提示两种模式。

08

消息解析

按 category 路由解析学生端聊天、窗口标题、WiFi 数量、浏览器 URL 与系统性能状态。

COMD subcmd 0x200 键鼠锁定控制（case 6 lock / case 0x0E unlock）

交互命令

list 列出已登录学生（含建档信息）

preview <ip> 请求屏幕缩略图

all 请求所有学生缩略图

info <ip> [0|1|2] 请求学生信息（系统/进程/窗口）

ps / wins <ip> 查看进程 / 窗口列表

blackscreen / bs <ip> 黑屏 + 锁键鼠

unlock <ip> 解锁黑屏与键鼠

bsall [lock] [text] 全员黑屏

shutdown / sd <ip> [秒] [文字] 远程关机

reboot / rb <ip> [秒] [文字] 远程重启

msg <ip> <text> 发送聊天消息
